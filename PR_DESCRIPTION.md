# PR: コレクション展覧会UIと登録フローの改善

## 概要
獲得したプライズを視覚的に楽しめる「展覧会（Gallery）」モードの実装と、ユーザーが商品を登録した直後にスムーズにコレクションに追加できるフローの整備を行いました。

## 変更点

### 1. コレクション展覧会UI (Gallery View)
- `CollectionScreen` にリスト表示とグリッド表示の切り替え機能を追加
- グリッド表示の実装 (`CollectionGridItem`)
- 詳細ダイアログとシェア機能の実装 (`share_plus` 利用)

### 2. 登録フローの改善 (Streamlined Registration)
- **商品登録画面**: 「これも獲得済みとして記録する」チェックボックスを追加し、登録と同時にコレクション追加可能に。
- **スキャン結果画面**: 既登録品が表示された際、「この商品をGETした！」ボタンからワンタップでコレクション追加可能に。

### 3. Firestore権限設定
- `users/{userId}/collections` への書き込み権限が不足していた問題を修正し、セキュリティルールを更新。
