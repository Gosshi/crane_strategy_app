# 著作権と埋め込み再生

YouTube動画の埋め込み再生は、YouTubeの公開機能（共有→埋め込み）を使ってストリーミング視聴させるものであり、動画ファイルをコピー・複製する行為ではありません。そのため、埋め込み自体は著作権侵害には当たらないと一般に解されます。実際、弁護士解説でも「リンクを貼って動画を埋め込む行為は、著作物の複製ではないため、原則として著作権侵害にならない」とされています。  
したがって、公開されている他人のYouTube動画を公式プレーヤーで再生するだけであれば、法律上の著作権許可は通常不要です。

ただし、以下の点には注意が必要です。

- 埋め込み対象の動画自体が、権利者許諾なく違法アップロードされたものである場合  
- 投稿者（権利者）が埋め込みを無効化している場合（技術的に再生不可）

---

# 公式プレイヤー使用時の収益化制限

YouTube利用規約では「個人的な非商用利用」を前提としつつ、「埋め込み型プレーヤーによる動画表示」は明示的に許可されています。

一方で、以下の行為は禁止・制限されています。

- YouTubeコンテンツ上への独自広告掲載  
- YouTube動画を主目的としたページ・アプリでの広告販売  
- YouTube APIデータを含む画面に対する広告やスポンサー販売  

YouTubeヘルプでも、

> 「ブログなどで動画を埋め込んで広告収益を得ようとするサイトは、利用規約違反と見なされる場合がある」

と明記されています。

### 実務上の解釈

- 一般的なブログやアプリで、**補助的に動画を紹介する程度** → 問題になりにくい  
- **大量のYouTube動画を主コンテンツとして並べ、広告収益を得る構成** → 規約違反リスクが高い  

公式プレイヤーを使用した場合、動画内広告の収益は **YouTubeと動画投稿者** に帰属します。  
アプリ開発者が動画自体の広告収益を得ることはできません。

そのため、商用アプリの場合は以下が重要になります。

- 動画は「補助的要素」に留める  
- アプリ独自の付加価値（解説、整理、検索、UI、独自ノウハウ）を明確に持たせる  

---

# 権利者の許可

通常の埋め込み再生では、以下の理由から **投稿者（権利者）の個別許可は不要** です。

- 動画ファイルを複製・再配布していない  
- YouTube利用規約により、公開動画は第三者埋め込みが許可されている  

ただし、以下の場合は不可です。

- 埋め込み不可設定の動画  
- 限定公開・メンバー限定動画  
- 明らかに違法アップロードされた動画  

アプリ側が **違法動画を積極的に紹介・集約する構成** は、リスクが高くなります。

# API利用上の注意点

YouTube API（iFrame API / Mobile SDK）を利用する場合、以下が必須です。

## 利用規約・表示義務

- アプリの利用規約・プライバシーポリシーに  
  - YouTube APIを利用していること  
  - YouTube利用規約へのリンク  
  を明記すること

-  
  > 「本アプリでYouTube APIを利用することにより、ユーザーはYouTube利用規約に同意したものとみなされる」
  
  旨の記載が推奨されます。

## 技術・UI要件

- プレイヤーサイズは最低 200×200px  
- コントロール、YouTubeロゴ、動画情報を削除・改変しない  
- YouTubeが配信する広告・関連動画をブロックしない  

---

# サブスクリプション・課金設計の注意

YouTube規約では、以下が問題になり得ます。

- **YouTube動画へのアクセス自体を有料にすること**
- 有料会員でないと動画が見られない構成

そのため、サブスクリプションを導入する場合は、

- 課金対象は  
  - 独自解説  
  - データベース機能  
  - おすすめロジック  
  - 攻略ノウハウ  
- YouTube動画は「参考資料」「補足情報」

という位置づけを明確にする必要があります。

---

# 運用上の留意点

## 広告配置

- 動画プレイヤーと広告を明確に分離する  
- 動画自体に広告を被せない  
- 「YouTube広告＝YouTube側管理」であることを尊重する  

## プレイヤーUI

- 自動再生・非表示操作などは公式に許可された範囲のみ使用  
- プレイヤー改変は禁止  

## 規約・ポリシー

- 利用規約・プライバシーポリシーを必ず整備  
- Google API / YouTube APIの利用条件を明示  

## リスク対応

- APIキー停止、アプリ削除の通知が来た場合は即対応  
- 特に「動画集約アプリ」は審査・監視対象になりやすい  

---

# 結論

- **他人のYouTube動画を公式プレイヤーで埋め込む行為自体は違反ではない**
- ただし、
  - 動画が主コンテンツ
  - 広告・課金と強く結びつく
  - 大量転載型アプリ
  は **規約違反リスクが高い**

**「動画は補助」「価値はアプリ側」**  
この構造を明確にすれば、iOS / Androidの商用アプリでも比較的安全に運用できます。
