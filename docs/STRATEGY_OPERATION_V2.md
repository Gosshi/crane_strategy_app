# 攻略コンテンツ運用ガイド V2 (Compliance First)

## 原則
**「自社権利物以外は掲載しない」**
運営は、権利関係がクリーンなコンテンツのみを公式情報として提供する。

## 動画URLの設定基準 (ホワイトリスト)
以下のいずれかに該当する場合のみ、`Strategy` データに `videoUrl` を設定できる。
1. **Self-Produced**: 運営チーム自身が撮影・編集・アップロードした動画。
2. **Official Partner**: 運営が直接許諾を得たパートナーの動画。
3. **Manufacturer**: メーカー（セガ、バンダイナムコ等）の公式プロモーション動画。

**※ 一般YouTuberの攻略動画は、いかに有益であっても運営が無断で紐付けてはならない。**

## 代替措置
動画がない場合は `videoUrl` を空にし、システム側で自動的に「攻略図解 (`StrategyDiagram`)」を表示させる。
