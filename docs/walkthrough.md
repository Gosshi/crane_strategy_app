# ウォークスルー: ゲーミフィケーション & YouTube連携

## 実装された機能

### 1. "GET!" 演出 (Confetti / 紙吹雪)
- **Confetti Overlay**: `confetti` パッケージを使用して、完了時の演出レイヤーを作成しました。
- **スキャン結果画面**: コレクションへの追加成功時に、達成感を演出する紙吹雪アニメーションを統合しました。

### 2. ランクシステム (ゲーミフィケーション)
- **UserLevelService**: 獲得数に基づいてユーザーランクを計算するサービスを作成しました。
  - **ビギナー**: 0-4 個
  - **クレーンゲーマー**: 5-19 個
  - **熟練の攻略家**: 20-49 個
  - **神の手**: 50個以上
- **アカウント画面**: 現在のステータスと次の目標を示す、グラデーション付きのランクバッジとプログレスバーを追加しました。

### 3. **YouTube攻略動画連携 (UGC)**
- **投稿機能**: 攻略情報の投稿時に、YouTubeのURLを入力できるようにしました。クリップボードからの**貼り付けボタン**も追加し、スマホでの入力負荷を軽減しました。
- **スキャン結果画面**: YouTubeリンクが含まれる投稿には「動画を見る (YouTube)」ボタンを表示し、タップすると外部アプリで動画を再生します。
- **動画詳細画面**: 誤操作防止のため、動画詳細画面からの「獲得」ボタンを削除し、獲得アクションはスキャン結果画面に一本化しました。

### 4. ビジュアル & ユーザビリティ改善
- **Neonテーマ**: より「ゲーミング」らしいスタイルにテーマを調整しました。
- **Pull-to-Refresh**: 獲得コレクション画面に「引っ張って更新」機能を追加しました。
- **自動ログイン**: 未ログイン時の投稿・コレクション追加フローを改善しました。

## 検証結果
- **アニメーション**: "GET!" 時に紙吹雪が舞うことを確認。
- **ランク**: 獲得数に応じてランク表記が変わることを確認。
- **YouTube連携**: 投稿画面でURLを入力し、詳細画面のボタンからYouTubeアプリ/ブラウザが起動することを確認。
