# 攻略図解コンテンツ 実装計画 (Visual Strategy)

## 目的
著作権リスクのない「自社資産」としての攻略情報を提供する。

## 技術アプローチ
Flutter `CustomPainter` を使用し、デバイスの解像度に依存しない軽量なベクター描画を行う。

## 実装パターン
1. **Bridge (橋渡し)**: 平行棒、景品箱、重心マーカー、アーム下降位置（矢印）。
2. **Kenzan (剣山)**: 台座、トゲ、景品配置。
3. **3-Claw (3本爪)**: シールド高、掴み/押しポイント。

## フォールバック挙動
詳細画面 (`DetailScreen`) において、`videoUrl` が未設定の場合は自動的に本Widgetを表示する。
